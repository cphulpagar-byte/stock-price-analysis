# 📈 Stock Price EDA & Prediction

## Overview
Analysis of 2 years of real NSE stock data for 5 major Indian companies — 
TCS, Infosys, Wipro, HDFC Bank, and Reliance — using Python.

## Key Highlights
- 📊 Analyzed 500+ trading days of real market data
- 🔍 EDA: price trends, volatility, daily returns, correlation heatmap
- 📉 Built 30-day & 90-day Moving Average indicators
- 🤖 Linear Regression model to predict next-day closing price
- 📏 Model Evaluation: MAE, RMSE, R² Score

## Tools & Technologies
Python · Pandas · NumPy · Matplotlib · Seaborn · Scikit-learn · yfinance

## Project Structure
- `Stock_Price_EDA_Prediction.ipynb` — Main notebook with full analysis
